```bash
#设置为linux终端
session-manager >> 设置[最右边的设置符号] >> Terminal >> Emulation >> Terminal
右侧下拉框选择 Linux

#启用默认颜色
session-manager >> 设置[最右边的设置符号] >> Terminal >> Emulation >> Terminal
最右侧单选框 ANSI Color 和 Use color scheme 全部勾选

#设置为【白+黑】面板格式/颜色
session-manager >> 设置[最右边的设置符号] >> Terminal >> Appearance >> Current Color scheme
选择 White/Black

#设置不同文件类型颜色格式
session-manager >> 设置[最右边的设置符号] >> Terminal >> Appearance >> ANSI Color >> Bold colors
自行每个位置颜色对应的文件类型，修改为自己喜欢的颜色
```
